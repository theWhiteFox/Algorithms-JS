Two Sum Algorithm

The two sum algorithm will be written in JavaScript(JS) that runs in O(n) time. The algorithm will be a JS function that take two parameters, a numbers array arr a sum value sum. This function returns a of pair indices from arr that add up to the sum parameter.

This first attempt is using brute force O(n2). To test go to your browser console and call the function by copy / paste twoSumBrute(arr, 10); The arr passed in has already been defined within the JS code. You could also enter an array and a sum of your own like this example twoSumBrute([9, 33, 50, 41], 59); If a sum is not in the array an error will the arr and sum passed in will be returned "Error no two numbers in array for sum: 159"

// Two Sum return indices O(n^2) const twoSumBrute = (arr, target) => { let len = arr.length; for (let i = 0; i < len; i++) { for (let j = i + 1; j < len; j++) { if (arr[j] === target - arr[i]) { return [i, j]; } } } return `Error no two numbers in ${arr} for sum: ` + target; }; const arr = [3, 7, 11, 15];
